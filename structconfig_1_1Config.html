Detailed Description

Configuration structure for application settings.

Manages application configuration parameters with default values. Provides methods for loading configurations from JSON files and command-line arguments parsing.

Prepare configuration from command-line arguments.

Parses command-line arguments to find configuration path. If path is provided via -c/–config, loads configuration from file. Otherwise returns default configuration.
