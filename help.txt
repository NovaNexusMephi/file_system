DIR 
Output of the catalog contents with the ability to filter and save the result.
Parameters: 
<full> [bool] - optional: display full files info. 
Keys: -

INIT
Format a disk to create a file system with specified parameters.
Parameters:
<volume_id> [string] - mandatory: volume label.
<owner> [string] - optional: owner name.
Keys:
/segm:N [int] - optional: number of catalog segments (default: calculated).
/rec:N [int] - optional: number of entries per segment.
/vol:N [int] - mandatory: volume label (redundant with <volume_id>).

CREATE
Create a new file with preallocated space.
Parameters:
<filename> [string] - mandatory: name of the file.
Keys:
/allocate:<size> [int] - mandatory: number of blocks to allocate.

DELETE
Delete a file.
Parameters:
<filename> [string] - mandatory: name of the file to delete.
Keys: -

RENAME
Rename a file, including its name and extension.
Parameters:
<old_name> [string] - mandatory: current file name.
<new_name> [string] - mandatory: new file name.
Keys: -

COPY
Copy a file within the file system.
Parameters:
<source> [string] - mandatory: source file name.
<destination> [string] - mandatory: target file name.
Keys: -

MOVE
Move a file, replacing the target file if it exists.
Parameters:
<source> [string] - mandatory: source file name.
<destination> [string] - mandatory: target file name.
Keys: -

ADD
Append data to the end of an existing file.
Parameters:
<filename> [string] - mandatory: name of the file.
Keys:
/size:<blocks> [int] - mandatory: number of blocks to add.
