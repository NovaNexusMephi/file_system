Detailed Description

Command to initialize the file system with specified parameters.

Handles file system formatting by configuring storage parameters and preparing the catalog structure. Follows the Command design pattern.

std::string InitCommand::execute ( )
overridevirtual

Executes the file system initialization.

The process includes:

  1. Check if already initialized
  2. Set system metadata (volume/owner)
  3. Initialize catalog structure
  4. Mark file system as valid
Returns
OK on success, ERROR if already initialized.
