Detailed Description

Command to display detailed free space information.

Shows locations and sizes of free blocks in the file system, including both allocated segment space and direct free space. Follows the Command pattern.

std::string FreeCommand::execute ( )
overridevirtual

Executes the free space reporting operation.

The output format includes:

  1. Numbered list of free block ranges
  2. Total occupied space
  3. Total free space

Example output: OK:

  1. 100 512
  2. 613 256 Occupied: 8192 Free: 1024
Returns
Formatted free space report or error message. Possible returns: OK with details, ERROR if uninitialized.
