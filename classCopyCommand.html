Detailed Description

Command to copy an existing file to a new location.

Creates a duplicate of the specified file with a new name. Requires the source file to exist and the destination name to be available. Uses CreateCommand internally to allocate space for the copy.

Executes the file copy operation.

Execution steps:

  1. Validate file system initialization
  2. Check destination name availability
  3. Verify source file existence
  4. Create new file with source's size using CreateCommand
